Introduction

Welcome to the Dreamy documentation. Dreamy is an AI-first web builder that generates semantic, accessible HTML & CSS based on natural language prompts.

How it works

Simply describe what you want to build in the prompt box. The AI analyzes your request, determines the best UX pattern, and generates the code.

$ npm install dreamy-cli -g $ dreamy init my-new-project > Generating magic... Done!

Why Dreamy?

Unlike other builders, we don't lock you in. You can export your project at any time to standard React, Vue, or HTML/CSS stacks.
